# Скрипт автоматической настройки VPS

Автоматический скрипт для первоначальной настройки VPS под VPN на базе Xray с протоколом VLESS + Reality.

## Что делает скрипт

*   Проверяет, что запущен от имени root
*   Определяет операционную систему (Ubuntu/Debian или CentOS/Rocky/AlmaLinux)
*   Обновляет систему и устанавливает необходимые пакеты (fail2ban, wget, jq, openssl)
*   Настраивает fail2ban для защиты SSH от брутфорс-атак
*   Устанавливает Xray последней версии
*   Генерирует ключи для Reality протокола (PrivateKey и PublicKey)
*   Создает базовый конфигурационный файл Xray с VLESS + Reality на порту 443
*   Запрашивает и устанавливает новый пароль для root
*   Создает нового обычного пользователя без sudo-прав
*   Запрещает вход по SSH для пользователя root
*   Выводит информацию для настройки клиентов (PublicKey, ShortID, порт, SNI)

## Требования

*   Свежеустановленная VPS с Ubuntu 20.04+, Debian 11+, CentOS 8+, Rocky Linux или AlmaLinux
*   Доступ по SSH под пользователем root
*   Минимум 512 MB RAM

## Использование

Скачайте скрипт на VPS:

`wget https://raw.githubusercontent.com/vpupkin123/The-Script/main/the-script.sh`

Сделайте его исполняемым:

`chmod +x the-script.sh`

Запустите от имени root:

`./the-script.sh`

Скрипт задаст несколько вопросов:

*   Новый пароль для root
*   Логин нового пользователя
*   Пароль нового пользователя

## Управление пользователями Xray

После установки используйте отдельный скрипт `xray.sh` для управления клиентами:

*   `./xray.sh add <username>` — добавить пользователя
*   `./xray.sh del <username>` — удалить пользователя
*   `./xray.sh list` — список пользователей
*   `./xray.sh link <username>` — получить ссылку для клиента
*   `./xray.sh help` — справка

## Особенности конфигурации

Скрипт создает конфигурацию VLESS + Reality со следующими параметрами:

*   Протокол: VLESS с flow xtls-rprx-vision
*   Порт: 443
*   Маскировка: трафик имитирует соединение с cloudflare.com
*   Автоматическая генерация уникальных ключей при каждом запуске

## Безопасность

*   SSH доступ для root отключен
*   Fail2ban блокирует IP после 5 неудачных попыток входа в течение 10 минут на 1 час
*   Новый пользователь создается без sudo-прав
